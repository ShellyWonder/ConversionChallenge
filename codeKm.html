ConvertMe

ConvertMe is a simple program converting the input integer from kilometers to miles and displaying the results to the page.

Brief code overview:

  • Initiation:
    • event listener (located in the bottom <script> tag) is triggered by a "click" on the "submit" button.

    • Input from the user field ("kmValue") is passed into the "convertItKm" function.

  • Validation:
    • Input checked to make certain it is an integer (number) and not a string.

    • If input is a number treated as a string, the program parses it to an integer.

    • If input is a text, the system will alert the user, "Please enter a number."

  • Logic:

    Within the "convertItKm" function, the "convertToKm" function is called:

    • the parameter "KmValue" is passed

    • the conversion equation is executed

  • Display:
    • Finally, results are published to the HTML via the "results" class.
